# HTML Versions

## 1. Introduction

HTML has gone through several versions since its creation. Each version introduced new features, improved standards, and addressed the growing needs of web developers and users. Understanding the different versions helps developers know which features are available and how to support older browsers when necessary.

---

## 2. Complete Version History

### HTML 1.0 (1993)

**Status:** First Draft

**Key Features:**
- Basic document structure
- Headings (`<h1>` to `<h6>`)
- Paragraphs (`<p>`)
- Hyperlinks (`<a>`)
- Lists (`<ul>`, `<ol>`, `<li>`)
- Horizontal rules (`<hr>`)

**Limitations:**
- No images
- No tables
- No forms
- No styling capabilities
- Very limited formatting options
