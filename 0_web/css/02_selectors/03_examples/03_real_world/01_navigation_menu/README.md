# Real-World Example: Navigation Menu

## Objective

Build a responsive **navigation menu** using CSS selectors.

This project demonstrates how different selectors work together in a practical website layout.

---

## Concepts Used

- Element Selectors
- Class Selectors
- Grouping Selectors
- Descendant Selectors
- Pseudo-class Selectors (`:hover`)
- Universal Selector
- CSS Reset

---

## Project Structure

- `index.html` → Navigation menu structure
- `style.css` → Navigation styles

---

## Features

- Modern navigation bar
- Logo section
- Navigation links
- Hover effects
- Call-to-action button
- Responsive layout foundation
- Clean and reusable CSS

---

## Learning Outcomes

After completing this project, you will understand:

- How selectors work together
- How to build a professional navigation bar
- How to organize reusable CSS
- How modern websites structure navigation menus
