# Universal Selector Example

## Objective

Learn how to use the **Universal Selector (`*`)** in CSS.

The Universal Selector selects **every HTML element** on a webpage. It is commonly used for creating CSS resets, applying global styles, and ensuring consistent layouts across browsers.

Example:

```css
*{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

The above CSS removes the default spacing from every HTML element.
