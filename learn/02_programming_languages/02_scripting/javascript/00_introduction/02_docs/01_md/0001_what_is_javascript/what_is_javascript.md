# What is JavaScript?

## Introduction

**JavaScript (JS)** is a high-level, interpreted, and dynamically typed programming language used to create interactive and dynamic web pages. It is one of the three core technologies of web development, alongside **HTML** and **CSS**.

- **HTML** provides the structure of a web page.
- **CSS** controls the appearance and layout.
- **JavaScript** adds functionality, behavior, and interactivity.

Today, JavaScript is one of the most popular programming languages in the world and is used for both **front-end** and **back-end** development.

---

# Why JavaScript?

Without JavaScript, websites would be mostly static.

JavaScript allows you to:

- Respond to user actions
- Update content without reloading the page
- Validate forms
- Create animations
- Communicate with servers
- Build games
- Develop mobile and desktop applications

---

# Example

## HTML

```html
<button onclick="showMessage()">Click Me</button>

<script>
function showMessage() {
    alert("Welcome to JavaScript!");
}
</script>
```

When the user clicks the button, JavaScript displays an alert message.

# What Can JavaScript Do?

JavaScript can perform many tasks, including:

- Change HTML content
- Change CSS styles
- Show or hide elements
- Handle mouse and keyboard events
- Perform calculations
- Validate user input
- Create animations
- Fetch data from APIs

# Features of JavaScript

Some important features include:

- High-level language
- Lightweight
- Dynamic typing
- Object-oriented
- Prototype-based
- Event-driven
- Multi-paradigm
- Cross-platform
- Interpreted

---

# Advantages

- Easy to learn
- Runs in all modern browsers
- Large community
- Massive library support
- Full-stack development
- Open-source
- Fast execution
- Platform independent

---

# Limitations

- Client-side code is visible to users.
- Browser security restrictions apply.
- Different browsers may have slight differences.
- Single-threaded execution (with asynchronous capabilities).
