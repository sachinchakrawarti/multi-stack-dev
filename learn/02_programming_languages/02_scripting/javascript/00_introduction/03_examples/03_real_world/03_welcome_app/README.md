# 03 - Welcome App

## 📖 Overview

The **Welcome App** is a beginner-friendly JavaScript project that accepts a user's name and displays a personalized welcome message.

It demonstrates how JavaScript can interact with HTML elements, process user input, and update webpage content dynamically.

---

## 📁 File Structure

```text
03_welcome_app/
│
├── index.html
├── README.md
├── script.js
└── style.css
```

---

## 🎯 Objectives

After completing this project, you will learn how to:

- Read user input
- Validate input
- Handle button click events
- Handle keyboard events
- Update HTML dynamically
- Use template literals
- Display personalized messages

---

## ▶️ How to Run

1. Open **index.html** in your browser.
2. Enter your name.
3. Click **Welcome Me** or press **Enter**.
4. View the personalized welcome message.
