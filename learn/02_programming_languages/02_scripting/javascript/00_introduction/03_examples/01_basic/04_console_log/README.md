# 04 - console.log()

## 📖 Overview

This example demonstrates how to use the **`console.log()`** method in JavaScript.

`console.log()` is one of the most commonly used functions for:

- Displaying information
- Debugging programs
- Testing variables
- Inspecting objects and arrays

## 🧪 Practice Exercises

### Exercise 1

Print your city.

```javascript
console.log("Bhopal");
```

---

### Exercise 2

Print your age.

```javascript
console.log(22);
```

---

### Exercise 3

Create an object.

```javascript
const employee = {
    name: "Rahul",
    department: "IT"
};

console.log(employee);
```

---

### Exercise 4

Print an array.

```javascript
const fruits = ["Apple", "Banana", "Mango"];

console.log(fruits);
```

---

### Exercise 5

Print the current time.

```javascript
console.log(new Date());
```

## 💡 Tips

- Use `console.log()` frequently while learning JavaScript.
- It helps you understand variable values and program flow.
- Remove unnecessary `console.log()` statements before deploying production code.
