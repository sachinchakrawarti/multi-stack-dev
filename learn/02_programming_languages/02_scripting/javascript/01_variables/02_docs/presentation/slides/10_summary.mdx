## Key Points

- Variables store data.
- Use meaningful variable names.
- Prefer `const` by default.
- Use `let` when values need to change.
- Avoid `var` in modern JavaScript.
- Initialize variables whenever possible.
- Follow JavaScript naming conventions.

## Quick Comparison

| Keyword | Redeclare | Reassign | Scope |
|---------|-----------|----------|-------|
| `var` | ✅ | ✅ | Function |
| `let` | ❌ | ✅ | Block |
| `const` | ❌ | ❌ | Block |
