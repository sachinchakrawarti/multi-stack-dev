# 01 - Greeting Page

## 📖 Overview

This project demonstrates a simple **Greeting Page** built with HTML, CSS, and JavaScript.

When the user clicks the button, JavaScript displays a random greeting message and updates the page dynamically.

---

## 📁 File Structure

```text
01_greeting_page/
│
├── index.html
├── README.md
├── script.js
└── style.css
```

---

## 🎯 Objectives

After completing this project, you will learn how to:

- Connect HTML, CSS, and JavaScript
- Handle button click events
- Update webpage content dynamically
- Display random messages
- Manipulate the DOM using JavaScript

---

## ▶️ How to Run

1. Open **index.html** in your browser.
2. Click the **Show Greeting** button.
3. Observe the greeting message.
4. Open the browser console (F12 → Console) to see the logged output.
