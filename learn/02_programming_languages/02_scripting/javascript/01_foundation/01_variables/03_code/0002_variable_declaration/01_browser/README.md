# Browser Demo - Variable Declaration

This is an interactive browser-based demonstration of JavaScript variable declaration using `var`, `let`, and `const`.

## Features

- **Interactive Buttons**: Click to see real-time examples
- **Visual Output**: Results displayed directly on the page
- **Styled UI**: Clean, modern design with color-coded sections
- **Console Integration**: Additional examples logged to browser console

## How to Use

1. Open `index.html` in any modern web browser
2. Click the buttons to run different variable declaration examples
3. Open Developer Tools (F12) to see console output

## What You'll Learn

### var
- Function scope behavior
- Hoisting
- Re-declaration
- Global object attachment

### let
- Block scope
- Temporal Dead Zone (TDZ)
- No re-declaration
- No global attachment

### const
- Must be initialized
- Cannot be reassigned
- Object/array mutability

## Color Coding

- 🟠 **Orange** - `var` examples
- 🟢 **Green** - `let` examples
- 🔴 **Red** - `const` examples
- 🟣 **Purple** - Comparison
