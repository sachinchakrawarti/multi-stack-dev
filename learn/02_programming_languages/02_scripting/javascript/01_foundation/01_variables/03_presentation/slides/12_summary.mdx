## Key Takeaways

- Variables store data.
- Use meaningful variable names.
- Prefer `const` whenever possible.
- Use `let` when values change.
- Avoid `var` in modern JavaScript.
- Understand block scope and function scope.
- Learn hoisting and the Temporal Dead Zone.
- Write clean, readable, and maintainable code.

---

## Best Practice Rule

> **Default to `const`.  
> Use `let` only if reassignment is required.  
> Avoid `var` in modern JavaScript development.**
