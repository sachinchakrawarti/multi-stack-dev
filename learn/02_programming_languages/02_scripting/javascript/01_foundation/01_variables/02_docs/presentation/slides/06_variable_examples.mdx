# Variable Examples

## Example 1: Store User Information

```javascript
let name = "Sachin";
let age = 24;
let city = "Bhopal";

console.log(name);
console.log(age);
console.log(city);
```

---

## Example 2: Shopping Cart

```javascript
let product = "Laptop";
let price = 55000;
let quantity = 2;

let total = price * quantity;

console.log(total);
```

Output

```text
110000
```

---

## Example 3: Game Score

```javascript
let score = 0;

score = score + 10;
score = score + 20;

console.log(score);
```

Output

```text
30
```

---

## Example 4: Student Details

```javascript
let student = {
    name: "Rahul",
    age: 20,
    course: "B.Tech"
};

console.log(student.name);
```

---

## Example 5: Boolean Variable

```javascript
let isLoggedIn = true;

console.log(isLoggedIn);
```

Output

```text
true
```

---

## Real-World Applications

Variables are commonly used for:

- User login information
- Shopping cart data
- Online banking balances
- Weather information
- Form inputs
- API responses
- Game scores
- Chat messages
