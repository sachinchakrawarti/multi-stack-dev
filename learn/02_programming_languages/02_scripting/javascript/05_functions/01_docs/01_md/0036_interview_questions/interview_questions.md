# Introduction

Functions are one of the most frequently tested topics in JavaScript interviews.

Interviewers ask questions ranging from basic syntax to advanced concepts like closures, callbacks, hoisting, recursion, and asynchronous functions.

This lesson contains commonly asked interview questions to help you review everything learned in this module.

# Tips for Interviews

- Master the difference between **Function Declarations**, **Function Expressions**, and **Arrow Functions**.
- Understand **Scope**, **Closures**, and **Hoisting** thoroughly.
- Practice writing recursive functions.
- Be comfortable explaining callbacks, promises, and async/await.
- Learn how `this` behaves in different contexts.
- Understand `bind()`, `call()`, and `apply()`.
- Write small, clean, and reusable functions.
- Practice coding without relying on autocomplete.
- Explain your thought process while solving coding problems.
- Review real-world use cases for every function concept.
