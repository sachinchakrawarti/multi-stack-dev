# Real-World Example: Login Form

## Objective

Build a modern **Login Form** using CSS selectors.

This project demonstrates how different CSS selectors are used together to create a clean and professional authentication page.

---

## Concepts Used

- Universal Selector
- Element Selectors
- Class Selectors
- Descendant Selectors
- Attribute Selectors
- Pseudo-class Selectors (`:hover`, `:focus`)
- CSS Reset

---

## Project Structure

- `index.html` → Login page
- `style.css` → Form styling

---

## Features

- Modern login card
- Username & Password fields
- Remember Me checkbox
- Forgot Password link
- Login button
- Responsive centered layout
- Input focus effects
- Clean UI

---

## Learning Outcomes

After completing this project, you will understand:

- How to style forms using CSS
- How to use attribute selectors
- How to style input focus states
- How CSS selectors work together in real applications
