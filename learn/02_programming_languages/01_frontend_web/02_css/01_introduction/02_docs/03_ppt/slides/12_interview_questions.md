## Q4. Recommended CSS?

---

## Q5. CSS Syntax

```css
selector{

property:value;

}
```

---

## Q6. What is a Selector?

A selector chooses the HTML element(s) to style.

Example

```css
h1{
}
```

---

## Q7. What is a Property?

Example

```css
color
```

---

## Q8. What is a Value?

Example

```css

```
