# Programming Concepts Reference

## Introduction

Programming is built on a set of fundamental concepts that are common across almost all programming languages. These concepts form the foundation for writing software, understanding how programs work, and solving real-world problems.

This reference provides a quick overview of the most important programming concepts that every programmer should know.

# Common Terminology

| Term | Meaning |
|------|---------|
| Program | A set of instructions executed by a computer |
| Programming | The process of writing computer programs |
| Algorithm | Step-by-step solution to a problem |
| Flowchart | Graphical representation of an algorithm |
| Pseudocode | Language-independent program logic |
| Variable | Named storage location |
| Function | Reusable block of code |
| Object | Collection of related data and behavior |
| Class | Blueprint for creating objects |
| Array | Ordered collection of elements |
| Module | Reusable unit of code |
| Compiler | Converts source code into machine code |
| Interpreter | Executes source code line by line |
| Runtime | Environment where a program executes |
| API | Interface for communication between software systems |
