# CSS Selector Priority (Specificity) Example

## Objective

Learn how **CSS Selector Priority (Specificity)** works.

When multiple CSS rules target the same HTML element, the browser decides which rule to apply based on **specificity**. The selector with higher specificity takes precedence.

Specificity Order (Highest → Lowest)

```
!important
↓

Inline Style

↓

ID Selector

↓

Class Selector

↓

Element Selector

↓

Universal Selector
```
