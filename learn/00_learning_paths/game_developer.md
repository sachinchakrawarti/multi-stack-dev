# Game Developer Learning Path

## 🎯 Path Overview
**Duration**: 8-12 months
**Difficulty**: Intermediate
**Prerequisites**: Programming fundamentals

## 📚 Learning Roadmap

### Phase 1: Programming (4-6 weeks)
- ✅ C++ Fundamentals
- ✅ Object-Oriented Programming
- ✅ Data Structures
- ✅ Graphics Programming

### Phase 2: Unity (8-10 weeks)
- ✅ Unity Basics
- ✅ 2D/3D Development
- ✅ Physics & Animation
- ✅ UI/UX in Games
- ✅ Scripting (C#)
- ✅ Optimization

### Phase 3: Unreal Engine (8-10 weeks)
- ✅ Unreal Basics
- ✅ Blueprints
- ✅ C++ in Unreal
- ✅ Level Design
- ✅ Lighting & Effects

### Phase 4: Game Design (4-6 weeks)
- ✅ Game Mechanics
- ✅ Storytelling
- ✅ Level Design
- ✅ Player Experience

### Phase 5: Projects (8-10 weeks)
- ✅ 2D Platformer
- ✅ 3D Adventure Game
- ✅ Puzzle Game
- ✅ Multiplayer Game

## 🛠️ Key Technologies
- **C++** - Language
- **Unity** - Game Engine
- **Unreal Engine** - Game Engine
- **Blender** - 3D Modeling
- **Photoshop** - Art/Textures
