# Data Scientist Learning Path

## 🎯 Path Overview
**Duration**: 8-12 months
**Difficulty**: Intermediate
**Prerequisites**: Basic programming & statistics knowledge

## 📚 Learning Roadmap

### Phase 1: Python Fundamentals (4-6 weeks)
- ✅ Python Basics
- ✅ NumPy
- ✅ Pandas
- ✅ Data Visualization

### Phase 2: Statistics & Math (4-6 weeks)
- ✅ Probability
- ✅ Statistics
- ✅ Linear Algebra
- ✅ Calculus

### Phase 3: Machine Learning (6-8 weeks)
- ✅ Supervised Learning
- ✅ Unsupervised Learning
- ✅ Model Evaluation
- ✅ Feature Engineering

### Phase 4: Deep Learning (6-8 weeks)
- ✅ Neural Networks
- ✅ TensorFlow/PyTorch
- ✅ NLP
- ✅ Computer Vision

### Phase 5: Big Data (4-6 weeks)
- ✅ Hadoop
- ✅ Spark
- ✅ SQL/NoSQL
- ✅ Data Pipelines

### Phase 6: Deployment & Projects (6-8 weeks)
- ✅ Model Deployment
- ✅ MLflow
- ✅ Real-world Projects
- ✅ Capstone Project

## 🛠️ Key Technologies
- **Python** - Language
- **Pandas/NumPy** - Data Processing
- **Scikit-learn** - ML Library
- **TensorFlow/PyTorch** - Deep Learning
- **Tableau/PowerBI** - Visualization
- **MLflow** - ML Lifecycle
