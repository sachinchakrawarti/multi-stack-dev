# Cybersecurity Glossary

## Common Terms
- **Ethical Hacking**: Authorized penetration testing
- **Penetration Testing**: Simulated cyber attacks
- **Vulnerability**: Weakness in a system
- **Exploit**: Code that takes advantage of vulnerabilities
- **Payload**: Malicious code delivered during an attack
- **C2 (Command & Control)**: Server controlling compromised systems
- **Phishing**: Social engineering attack via email
- **Ransomware**: Malware that encrypts files for ransom
- **Zero-day**: Unknown vulnerability
- **Patching**: Fixing security vulnerabilities
