# Learning Path Guide

## Prerequisites
- Basic computer knowledge
- Command line familiarity
- Willingness to learn

## Recommended Order
1. Start with 01-08 (Fundamentals)
2. Move to 09-14 (Offensive)
3. Explore 15-22 (Advanced)
4. Practice with 23-30 (Specialization)

## Time Estimate
- Fundamentals: 2-3 months
- Offensive: 3-4 months
- Advanced: 4-6 months
- Specialization: Ongoing

## File Numbering
Each file has a unique number (F001-FXXX) for easy reference.
