# Learning Roadmap

## Phase 1: Fundamentals (Modules 1-8)
- 01_introduction - Basics of hacking
- 02_computer_fundamentals - Computer basics
- 03_linux - Linux OS
- 04_windows - Windows OS
- 05_networking - Network fundamentals
- 06_programming - Programming languages
- 07_web_technologies - Web basics
- 08_cryptography - Encryption & hashing

## Phase 2: Offensive Security (Modules 9-14)
- 09_reconnaissance - Information gathering
- 10_scanning - Port & service scanning
- 11_vulnerability_assessment - Finding vulnerabilities
- 12_web_security - Web app security
- 13_wireless_security - WiFi security
- 14_system_hacking - OS exploitation

## Phase 3: Advanced Topics (Modules 15-22)
- 15_malware_analysis - Malware examination
- 16_digital_forensics - Investigation
- 17_incident_response - Response procedures
- 18_cloud_security - Cloud platforms
- 19_mobile_security - Mobile devices
- 20_reverse_engineering - Code analysis
- 21_red_team - Offensive operations
- 22_blue_team - Defensive operations

## Phase 4: Specialization (Modules 23-30)
- 23_tools - Tool mastery
- 24_ctf - Capture the flag
- 25_certifications - Certification prep
- 26_projects - Practical projects
- 27_interview_questions - Interview prep
- 28_cheat_sheets - Quick references
- 29_resources - Learning materials
- 30_labs - Hands-on labs
