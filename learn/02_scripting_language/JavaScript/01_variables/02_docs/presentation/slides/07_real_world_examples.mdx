# Real-World Examples

## Variables in Everyday Applications

Almost every JavaScript application uses variables to store and manage data.

---

## Example 1: User Login

```javascript
let username = "Sachin";
let password = "********";
let isLoggedIn = true;
```

Variables store user credentials and login status.

---

## Example 2: Shopping Cart

```javascript
let product = "Laptop";
let quantity = 2;
let price = 55000;

let total = quantity * price;
```

Output

```text
110000
```

---

## Example 3: Banking Application

```javascript
let accountHolder = "Rahul";
let balance = 25000;

balance = balance + 5000;

console.log(balance);
```

Output

```text
30000
```

---

## Example 4: Social Media

```javascript
let likes = 120;
let comments = 45;
let shares = 12;
```

Variables keep track of user engagement.

---

## Example 5: Weather App

```javascript
let city = "Bhopal";
let temperature = 31;
let weather = "Sunny";
```

---

## Real-Life Analogy

Think of a variable as a **labeled storage box**.

```text
+--------------------+
| Username           |
|--------------------|
| Sachin             |
+--------------------+
```

The **label** is the variable name, and the **content** is the value.
