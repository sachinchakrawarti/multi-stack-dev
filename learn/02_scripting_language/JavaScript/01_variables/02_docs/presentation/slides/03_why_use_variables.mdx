# Why Use Variables?

## Variables Make Programs Dynamic

Variables allow your program to store and update information while it is running.

---

### Without Variables

```javascript
console.log("Sachin");
console.log("Sachin");
console.log("Sachin");
```

Every change requires editing the code.

---

### With Variables

```javascript
let name = "Sachin";

console.log(name);
console.log(name);
console.log(name);
```

Change the value once, and it updates everywhere.

---

## Common Uses

- Store user information
- Store scores
- Store prices
- Store API responses
- Store form data

---

## Key Takeaway

Variables make code reusable, flexible, and easier to maintain.
