## Definition

A **variable** is a named container used to store data in memory. It allows a program to save, update, and retrieve values whenever needed.

---

## Why Do We Need Variables?

- Store user input
- Save calculation results
- Reuse values
- Make programs dynamic
- Improve code readability

---

## Real-Life Analogy

Think of a variable as a **labeled box**.

📦 Box Label → `age`

Inside the box → `25`

Whenever you need the value, you simply use the label.

---

## Example

```javascript
let age = 25;
let name = "Sachin";

console.log(age);
console.log(name);
```

Output

```
25
Sachin
```

---

### Key Points

- Variables have a **name**
- Variables store a **value**
- Values can change (except `const`)
- Variables make programs reusable
