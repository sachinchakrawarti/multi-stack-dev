## What is Variable Declaration?

Variable declaration tells JavaScript to create a new variable in memory.

JavaScript provides three keywords:

- `var`
- `let`
- `const`

---

## Syntax

```javascript
let variableName = value;
```

General form

```javascript
keyword variableName = value;
```

---

## Examples

```javascript
let name = "Sachin";

const country = "India";

var age = 25;
```

---

## Declaration vs Initialization

Declaration

```javascript
let city;
```

Initialization

```javascript
city = "Bhopal";
```

Declaration + Initialization

```javascript
let city = "Bhopal";
```

---

## Variable Naming Rules

✅ Allowed

```javascript
let studentName;
let totalMarks;
let _user;
let $price;
```

❌ Not Allowed

```javascript
let 123name;
let user-name;
let let;
```

---

## Remember

- Use meaningful names
- Use camelCase
- Prefer `const`
- Use `let` when the value changes
- Avoid `var` in modern JavaScript
