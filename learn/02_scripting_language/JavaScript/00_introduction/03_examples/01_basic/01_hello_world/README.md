# 01 - Hello World

## 📖 Overview

This example demonstrates how to create and run your **first JavaScript program** using an external JavaScript file.

It also shows how to connect:

- HTML
- CSS
- JavaScript

into a simple web page.

---

## 📁 File Structure

```text
01_hello_world/
│
├── index.html
├── README.md
├── script.js
└── style.css
```

---

## 🎯 Objectives

After completing this example, you will learn how to:

- Create an HTML page
- Link an external CSS file
- Link an external JavaScript file
- Print messages using `console.log()`
- Display popup messages using `alert()`
- Handle a button click
- Open the browser console

---

## ▶️ How to Run

1. Open **index.html** in any modern web browser.
2. Press **F12** (or **Ctrl + Shift + I**).
3. Open the **Console** tab.
4. View the console output.
5. Click the **Click Me** button.

---

## 🧪 Practice Exercises

### Exercise 1

Print your name.

```javascript
console.log("Sachin");
```

---

### Exercise 2

Print your city.

```javascript
console.log("Bhopal");
```

---

### Exercise 3

Create another button that displays:

```text

```

---

### Exercise 4

Change the page heading using JavaScript.

```javascript
document.querySelector("h1").textContent =
    "Learning JavaScript";
```

---

### Exercise 5

Print today's date.

```javascript

```
