# Best Practices Recap

Follow these habits while writing JavaScript:

- Use `const` by default.
- Use `let` when reassignment is required.
- Avoid `var` in modern code.
- Use meaningful variable names.
- Write small, reusable functions.
- Keep your code organized.
- Use `===` instead of `==`.
- Handle errors properly.
- Follow consistent formatting.
- Practice regularly.

---

# Common Mistakes to Avoid

Avoid these beginner mistakes:

- Forgetting to declare variables
- Using `var` unnecessarily
- Using `==` instead of `===`
- Misspelling keywords
- Forgetting quotes around strings
- Modifying `const` variables
- Writing long, complex functions
- Repeating code
- Ignoring browser console errors
- Not validating user input

Learning from these mistakes will improve your coding skills.
