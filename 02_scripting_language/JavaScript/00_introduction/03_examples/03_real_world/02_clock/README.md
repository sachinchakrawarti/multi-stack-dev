# 02 - Digital Clock

## 📖 Overview

This project demonstrates how to build a **Digital Clock** using JavaScript.

The clock updates every second using `setInterval()` and displays the current time and date. Users can also switch between **12-hour** and **24-hour** time formats.

## ▶️ How to Run

1. Open **index.html** in your browser.
2. Observe the live digital clock.
3. Click **Toggle 12/24 Hour Format** to switch the display format.
