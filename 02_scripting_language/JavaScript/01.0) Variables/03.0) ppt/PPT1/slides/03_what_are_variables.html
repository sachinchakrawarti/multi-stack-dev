📦 What is a Variable?

A variable is a named container for storing data in memory.

📦 Storage Box Analogy

  • Label = Variable name (age)
  • Contents = Value (25)
  • Box = Memory address
const age = 25;
📦 age (Label) ➡️ 📝 25 (Value)

⚡ Why Variables?

  • 🔹 Reusability — Use values multiple times
  • 🔹 Readability — Human-readable names
  • 🔹 Maintainability — Update in one place
  • 🔹 Memory Management — Efficient storage

💡 Example: Instead of typing "John Doe" 10 times, use const name = "John Doe"

📌 Key Takeaway: Variables make code dynamic, reusable, and maintainable
