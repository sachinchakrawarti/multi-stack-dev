# Multiple Classes Example

## Objective

Learn how to apply **multiple classes** to a single HTML element.

An HTML element can have more than one class by separating class names with spaces. This allows you to combine reusable styles instead of creating new classes for every variation.

Example:

```html
<button class="btn primary large">
    Save
</button>
```

The button receives styles from all three classes:

- `.btn`
- `.primary`
- `.large`
