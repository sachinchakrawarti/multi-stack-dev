# CSS Reset Example

## Objective

Learn what a **CSS Reset** is and why it is one of the first things added to modern CSS projects.

Different browsers apply different default styles (margins, padding, font sizes, etc.). A CSS Reset removes these default styles so your website looks consistent across browsers.

Example:

```css
*{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

This is the most commonly used CSS Reset.
