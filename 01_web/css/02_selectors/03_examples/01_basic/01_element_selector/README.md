# Element Selector Example

## Objective

Learn how to use the **Element Selector** in CSS.

An Element Selector targets HTML elements by their tag name.

Example:

```css
h1{
    color: blue;
}
```

The above CSS styles every `<h1>` element on the page.

---

## Files

- `index.html` → HTML structure
- `style.css` → CSS styles

## Selectors Used

- `body`

- `h2`
- `p`
- `button`
- `ul`
- `li`
