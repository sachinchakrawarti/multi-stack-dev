# ID Selector Example

## Objective

Learn how to use the **ID Selector** in CSS.

An ID Selector targets a **single unique HTML element** using its `id` attribute. Unlike classes, an ID should only be used once per HTML document.

Example:

```css
#header{
    background: royalblue;
    color: white;
}
```

The above CSS styles only the element whose `id` is `header`.

After completing this example, you will understand:
