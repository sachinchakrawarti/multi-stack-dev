# Class Selector Example

## Objective

Learn how to use the **Class Selector** in CSS.

A Class Selector targets HTML elements using the `class` attribute. It is the most commonly used selector because it allows styles to be reused across multiple elements.

Example:

```css
.button{
    background: royalblue;
    color: white;
}
```

The above CSS styles every element with the class `button`.
