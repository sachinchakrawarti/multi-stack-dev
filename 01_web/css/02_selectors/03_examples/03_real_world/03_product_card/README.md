# Real-World Example: Product Card

## Objective

Build a modern **Product Card** using CSS selectors.

A Product Card is one of the most common UI components in e-commerce websites. This example demonstrates how various CSS selectors work together to create a clean, reusable card layout.

---

## Concepts Used

- Universal Selector
- Element Selectors
- Class Selectors
- Descendant Selectors
- Pseudo-class Selectors (`:hover`)
- CSS Reset
- Flexbox
- Box Shadow
- Border Radius

---

## Project Structure

- `index.html` → Product card structure
- `style.css` → Product card styling

---

## Features

- Product Image
- Product Title
- Category
- Product Description
- Price
- Rating
- Buy Now Button
- Hover Animation
- Responsive Card Layout

---

## Learning Outcomes

After completing this project, you will understand:

- How to build reusable UI components
- How selectors work together
- How to style buttons and images
- How hover effects improve user experience
