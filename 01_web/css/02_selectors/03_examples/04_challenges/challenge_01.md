# Challenge 01: Style a Simple Webpage

## Objective

Practice using **Element Selectors**.

---

## Requirements

Create a webpage containing:

- One `<h1>`
- Two `<h2>`
- Three `<p>`
- One `<button>`

Apply the following styles:

- `h1` → Blue color
- `h2` → Green color
- `p` → Gray text
- `button` → Blue background with white text

---

## Expected Skills

- Element Selector
- Basic CSS Syntax

---

## Bonus Challenge

Add a hover effect to the button.
