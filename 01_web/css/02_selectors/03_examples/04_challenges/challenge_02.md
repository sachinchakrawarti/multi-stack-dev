# Challenge 02: Personal Profile Card

## Objective

Practice using **Class Selectors**.

---

## Requirements

Create a profile card containing:

- Profile image
- Name
- Job title
- Short description
- Follow button

Use classes such as:

- `.card`
- `.profile-image`
- `.name`
- `.title`
- `.description`
- `.btn`

---

## Expected Skills

- Class Selector
- Reusable CSS
- Box Model

---

## Bonus Challenge

Add rounded corners and a shadow to the card.
