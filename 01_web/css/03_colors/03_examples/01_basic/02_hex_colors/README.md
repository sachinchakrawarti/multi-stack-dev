# CSS HEX Colors Example

## Objective

Learn how to use **HEX color codes** in CSS for styling text and backgrounds.

---

## Topics Covered

- HEX Color Format
- Text Color
- Background Color
- Color Consistency

---

## How to Run

1. Open `index.html` in your web browser.
2. Observe how different HEX color codes are applied to text and boxes.

---

## HEX Colors Used

| Color | HEX Code |
|--------|----------|
| Blue | `#1565C0` |
| Green | `#2E7D32` |
| Yellow | `#F9A825` |
| Red | `#D32F2F` |
| Gray | `#455A64` |

## What You Will Learn

After completing this example, you will be able to:

- Use HEX color values in CSS.
- Apply colors to text and backgrounds.
- Understand the structure of HEX color codes.
- Build consistent color palettes using HEX values.
