## RGB Colors Used

| Color | RGB Value |
|--------|-----------|
| Red | `rgb(255, 0, 0)` |
| Green | `rgb(0, 128, 0)` |
| Blue | `rgb(0, 0, 255)` |
| Purple | `rgb(128, 0, 128)` |
| Orange | `rgb(255, 165, 0)` |
| Gray | `rgb(96, 125, 139)` |

---

## RGB Value Range

Each RGB component can have a value between:

```text
0 to 255
```

Example:

```css
color: rgb(255, 0, 0);
```

- Red = 255
- Green = 0
- Blue = 0
