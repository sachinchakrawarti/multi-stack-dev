# Landing Page - CSS Colors Project

## Objective

Build a simple landing page that demonstrates the practical use of CSS colors in a real-world website.

---

## Topics Covered

- Color Palette
- Gradient Background
- Navigation Bar
- Hero Section
- Feature Cards
- Hover Effects
- Responsive Grid

---


## How to Run

1. Open `index.html` in any web browser.
2. Explore the landing page layout and observe how colors enhance the design.

---

## Features

- Responsive navigation bar
- Modern gradient hero section
- Call-to-action button
- Three feature cards
- Consistent color palette
- Hover effects
- Clean layout

## CSS Concepts Used

- Named Colors
- HEX Colors

- Background Colors
- Text Colors
- Border Radius

## Learning Outcome

After completing this project, you will be able to:

- Build a simple landing page using HTML and CSS.
- Apply a consistent color palette.
- Use gradients to create attractive backgrounds.
- Improve UI with shadows, spacing, and hover effects.
- Organize content into responsive sections.
