# History of CSS

> **CSS (Cascading Style Sheets)** was created to separate the presentation (style) of web pages from their structure (HTML). Before CSS, web developers had to use HTML tags for styling, making web pages difficult to maintain.

# Introduction

When the World Wide Web was first introduced, HTML was designed only to describe the **structure** of a document.

HTML could define:

- Headings
- Paragraphs
- Lists
- Tables
- Links

It was **not designed for styling**.

As websites became more popular, developers wanted colorful pages, different fonts, layouts, and better user experiences.

This led to the creation of CSS.

---

# The Web Before CSS

Before CSS existed, styling was done directly in HTML.

Example:

```html
<font color="red" size="6">
    Welcome
</font>
```

Developers also used:

- `<font>`
- `<center>`
- `<b>`
- `<i>`
- `<u>`
- Background attributes

Example:

```html
<body bgcolor="yellow">
```

Problems:

- Repeated code
- Difficult maintenance
- Large HTML files
- Poor readability
- No separation of concerns

---

# Why CSS Was Created

CSS was introduced to solve these problems.

Its goals were:

- Separate content from design
- Make websites easier to maintain
- Reduce duplicate code
- Allow consistent styling
- Improve browser rendering
- Support different devices

---

# Birth of CSS

CSS was first proposed by:

## Håkon Wium Lie

In **1994**, while working at **CERN**, Håkon Wium Lie suggested creating a separate language for styling web pages.

Later, **Bert Bos** joined the project, and together they developed CSS.

---

# CSS1 (1996)

Released by:

**World Wide Web Consortium (W3C)**

Features:

- Fonts
- Colors
- Backgrounds
- Margins
- Padding
- Borders
- Text alignment

CSS1 was the first official CSS recommendation.

---

# CSS2 (1998)

CSS2 introduced many advanced features.

New features included:

- Positioning
- Z-index
- Media types
- Tables
- Bidirectional text
- Better font handling

CSS2 made complex webpage layouts possible.

---

# CSS2.1 (2011)

CSS2.1 improved CSS2 by:

- Fixing browser inconsistencies
- Removing unstable features
- Improving compatibility
- Standardizing implementations

Many modern browsers fully support CSS2.1.

---

# CSS3

Unlike previous versions, CSS3 was **not released as a single specification**.

Instead, it was divided into independent **modules**.

Examples:

- Flexbox
- Grid
- Animations
- Transitions
- Media Queries
- Border Radius
- Shadows
- Gradients
- Transforms
- Multiple Backgrounds

This modular approach allows browsers to implement features independently.

---

# CSS Today

Modern CSS continues to evolve.

Some recent features include:

- CSS Variables
- Container Queries
- Logical Properties
- CSS Nesting
- Subgrid
- Cascade Layers
- Scroll Snap
- Aspect Ratio
- Modern Color Functions

CSS is continuously improved by the W3C and browser vendors.

# Evolution of Web Design

### Before CSS

```
HTML
 ├── Content
 ├── Layout
 ├── Colors
 ├── Fonts
 └── Styling
```

Everything was mixed together.

---

### After CSS

```
HTML
 └── Structure

CSS
 └── Presentation

JavaScript
 └── Behavior
```

Each technology has a specific responsibility.

---

# Why CSS Was Revolutionary

CSS changed web development by:

- Separating structure and style
- Making websites reusable
- Supporting responsive design
- Improving accessibility
- Reducing development time
- Making maintenance easier

---

# Interesting Facts

- CSS stands for **Cascading Style Sheets**.
- The word **"Cascading"** refers to how style rules are prioritized and inherited.
- CSS is maintained by the **World Wide Web Consortium (W3C)**.
- Every major web browser supports CSS.

---

# Advantages of Modern CSS

- Cleaner code
- Faster development
- Better performance
- Responsive layouts
- Rich animations
- Reusable stylesheets
- Easier maintenance
- Better user experience

---

# Summary

CSS transformed web development by separating presentation from structure. From its proposal in 1994 to today's powerful modular features like Flexbox, Grid, Variables, and Container Queries, CSS has become one of the three core technologies of the modern web alongside HTML and JavaScript.

As the web continues to evolve, CSS also continues to grow, enabling developers to build responsive, accessible, and visually appealing websites.
