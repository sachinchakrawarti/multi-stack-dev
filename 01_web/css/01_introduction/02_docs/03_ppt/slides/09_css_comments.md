# CSS Comments

---

# Syntax

```css
/* This is a comment */
```

---

# Why Comments?

- Explain code

- Organize sections

- Help teammates

- Improve readability

---

# Example

```css
/* Navigation */

.navbar{

background:#1565c0;

}
```

---

# Large Projects

```css
/*==================

Global Styles

===================*/

/*==================

Header

===================*/

/*==================

Footer

===================*/
```

---

# Bad Practice

CSS does NOT support

```css
// comment
```

Always use

```css
/* comment */
```

---

# Best Practices

✔ Explain WHY

✔ Organize Sections

✔ Remove Old Comments

✔ Keep Comments Short
