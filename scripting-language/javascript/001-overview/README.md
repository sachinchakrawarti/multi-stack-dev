# 001 - JavaScript Overview

> **Module 01** of the JavaScript Learning Roadmap

---

# 📖 Module Overview

This module introduces JavaScript, its history, features, use cases, runtime environments, and how to write and execute your first JavaScript program.

After completing this module, you will understand what JavaScript is, why it is one of the most popular programming languages, and where it is used in modern software development.

---

# 🎯 Learning Objectives

By the end of this module, you will be able to:

- Explain what JavaScript is.
- Understand the history of JavaScript.
- Identify where JavaScript is used.
- Describe the features of JavaScript.
- Understand how JavaScript works.
- Differentiate Browser JavaScript and Node.js.
- Write and run your first JavaScript program.
- Understand JavaScript's role in modern web development.

---

# 📚 Prerequisites

There are no prerequisites.

This module is designed for complete beginners.
