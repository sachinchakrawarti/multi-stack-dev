# 💡 Tips for Learning from Videos

- Watch actively instead of passively.
- Pause and type every code example yourself.
- Experiment by modifying the code.
- Take notes while watching.
- Build a small project after each major topic.
- Revise concepts regularly.
- Use official documentation to deepen your understanding.
