# What is JavaScript?

**JavaScript** is a high-level, interpreted programming language that conforms to the ECMAScript specification.

## 🎯 Key Characteristics

- ✅ **Multi-paradigm**: Object-oriented, functional, imperative
- ✅ **Dynamic typing**: Flexible variable types
- ✅ **First-class functions**: Functions are values
- ✅ **Prototype-based**: Inheritance through prototypes
- ✅ **Event-driven**: Responds to user interactions
- ✅ **Asynchronous**: Non-blocking I/O operations

## 🌍 Where is it Used?

- **Web Development**: Frontend & Backend
- **Mobile Apps**: React Native, NativeScript
- **Desktop Apps**: Electron, NW.js
- **IoT**: Node.js on Raspberry Pi
- **AI/ML**: TensorFlow.js, Brain.js

# History of JavaScript

## Evolution Timeline

| Year | Event | Impact |
|------|-------|--------|
| 1995 | Created by Brendan Eich | Born in 10 days at Netscape |
| 1996 | Microsoft JScript | Internet Explorer implementation |
| 1997 | ECMAScript 1 | First standard specification |
| 2009 | Node.js Released | JavaScript on the server |
| 2015 | ES6 (ECMAScript 2015) | Major language overhaul |
| 2024 | ECMAScript 2024 | Latest features |

## 🔑 Key Milestones

- **1997**: DOM Level 1 introduced
- **2005**: AJAX popularized
- **2009**: Node.js + CommonJS
- **2010**: jQuery dominates
- **2013**: React released
- **2014**: AngularJS 1.0
- **2016**: Vue.js 2.0
- **2020**: ES2020 with optional chaining
- **2024**: Temporal API and decorators
